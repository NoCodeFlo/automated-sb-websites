Project prompt: Rebuild “Pures Leben” website

1) Target URL to rebuild
- Base domain: http://www.pures-leben.ch/
- Rebuild the pages listed below using the existing content, structure, tone, and imagery.

2) Overview and goals
- Goal: Produce a modern, clean, warm, and calm redesign that improves readability, accessibility, and mobile responsiveness while preserving the existing content, site structure, and overall design language (colors, mood, and layout logic).
- Reuse all provided copy verbatim (do not paraphrase or invent new content).
- Reuse the existing image assets via their hosted URLs (provided below).
- Maintain existing navigation labels and footer legal links.
- Keep the visual language consistent with the current site: soft greens, warm gold accents, white backgrounds, generous whitespace, serif body text, script display headings.
- Improve usability through clear hierarchy, consistent spacing, accessible color contrast, and simplified layouts.
- If there are any existing external links (e.g., partner sites, email addresses), include them as external links. If any social links exist on any page, include them as external links only; do not attempt to replicate social content.

3) Global implementation guidelines
- Fonts:
  - Headings: Dancing Script (display)
  - Body: EB Garamond
  - Fallbacks: Georgia, Times, serif for body; cursive fallback for script
- Color tokens (define as CSS variables):
  - --color-black: #000000
  - --color-white: #FFFFFF
  - --color-shade1: #704244
  - --color-shade2: #b7cdb1
  - --color-main: #aaa43b
  - --color-secondary: #ccb04c
- Global components:
  - Header: left-aligned logo image, right-aligned nav with dropdown for subpages on “Home” and “Angebot & Preise”. Sticky on scroll for desktop; hamburger menu for mobile.
  - Footer: include links “Impressum”, “Datenschutz”, “Cookie-Einstellungen” (link to the existing URLs on the current site if available; else keep as placeholder anchors for now).
  - Cookie banner: simple dismissible banner with localStorage consent flag; links to “Datenschutz”.
- Typographic scale and spacing:
  - One h1 per page, logical h2/h3 hierarchy after that.
  - Comfortable line length (55–75 characters) and line-height (1.6 body).
  - Generous section spacing (min 56–72px on desktop; 32–40px mobile).
- Buttons/links:
  - Primary buttons: black background (#000000), white text; 12px–16px border radius; clear focus states.
  - Links in body copy: underlined; ensure contrast against background.
- Accessibility:
  - Provide meaningful semantic structure; use nav/ul for menus; main/section/article/aside where relevant.
  - Ensure contrast: if text rests on colored backgrounds (e.g., #ccb04c), use a white card/container or dark text to meet WCAG AA (4.5:1).
  - Keyboard accessible menus, accordions, and focus outlines.
  - Images must include alt attributes. Use the given note “No alt text available.” if no alt was provided.
  - Include a “Skip to content” link.
- Responsiveness & performance:
  - Mobile-first CSS; test at 360px, 768px, 1024px, 1280px breakpoints.
  - Use modern CSS (flex/grid) for layout.
  - Add width/height to images to prevent layout shift; loading="lazy"; decoding="async".
- Navigation structure (labels only; use the same labels and hierarchy):
  - Home
    - Über mich
    - Kontakt & Team
  - Ernähungsberatung
  - Massage & Schmerztherapie
  - Coaching & Hypnose
  - Angebot & Preise
  - Gruppenkurse
  - Aktuelles
  - For pages not in this brief, include the menu items but route them to their existing page URLs on the current site if available, or leave as non-functional placeholders (no new content).
